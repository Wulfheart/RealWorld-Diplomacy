Informationen zu RealWorld und der World-Variante

RealWorld ist eine Erweiterung von Realpolitik, die zum Auswerten von Partien mit
Wings verwendet werden kann. Außerdem wurde ein Teil der Sonderregeln der
Hanse-Variante umgesetzt.

RealWorld besitzt alle Funktionen von Realpolitik, aber (noch) keine eigene
Installationsroutine. Es ist daher für die Verwendung von RealWorld notwendig,
zuvor Realpolitik zu installieren.

Eine Alpha-Version mit Installationsroutine ist inzwischen verfügbar,
aber wie gesagt: ALPHA-Version...

Das Programm kann Standard-.dpy-Files lesen, erzeugt aber eigene Spielstände mit
der Endung .dpw, die von RP nicht gelesen werden können (und dort zu einer
Fehlermeldung führen). Dies ist notwendig, weil für die World-Variante einige
Zusatzinfos abgespeichert werden müssen. Es gibt aber eine Möglichkeit zum
Abspeichern im .dpy-Format, falls keine RealWorld-spezifischen Features genutzt werden.

Die wesentlichen Unterschiede zu Realpolitik sind die Möglichkeit der Verwendung
von Wings und die Option, auch Landfelder, die keine VZ sind, einer Nation
zuzuordnen und entsprechend einzufärben. Außerdem werden einige zusätzliche Aufbauoptionen
und Zugmöglichkeiten geschaffen, wie z.B. der Inselsprung von Armeen ohne Rückzugsmöglichkeit
(ab Version 1.8.9).

Ab Version 1.8.4 werden Kanäle bei der Auswertung auch regeltechnisch berücksichtigt.
